# Conversor Rural

¡Bienvenido al repositorio de mi juego de comercio y supervivencia rural! Este proyecto simula un sistema de economía basado en el trueque, situado en el corazón de un pequeño pueblo.

## Descripción

El juego se desarrolla en la **plaza central del pueblo**, justo al lado del edificio de comercio. Allí encontrarás a un **vecino comerciante**, un personaje clave con el que podrás interactuar para conseguir recursos esenciales para tu granja.

El sistema económico se basa exclusivamente en el **trueque**. La moneda de cambio principal es la **leña**, la cual deberás intercambiar por diversos animales y alimentos.

## Mecánicas de Juego

### El Ciclo de Juego
1.  **Inicio:** Comienzas la partida con **0 kg de leña**.
2.  **Recolección:** Debes dirigirte a las afueras de la plaza, hacia el bosque, para buscar árboles.
3.  **Tala:** Por cada árbol que tales, obtendrás **10 kg de leña**.
4.  **Comercio:** Regresa a la plaza y habla con el vecino para canjear tu leña por bienes.

* Gallinas
* Cabras

* Huevos
* Patatas

## Controles

El juego está diseñado para ser intuitivo. Aquí tienes la lista de controles para manejarte por el mundo y gestionar las negociaciones:

| Botón | Acción |
| :--- | :--- |
| **Flechas de Dirección** | Moverse por la plaza y el bosque. |
| **Botón A** | **Aceptar / Acción principal:** Sirve para iniciar el comercio, confirmar un trueque o talar un árbol. |
| **Botón B** | **Conversación:** Utilízalo para avanzar en los diálogos, seleccionar el tipo de animal y determinar la cantidad exacta que deseas adquirir. |
